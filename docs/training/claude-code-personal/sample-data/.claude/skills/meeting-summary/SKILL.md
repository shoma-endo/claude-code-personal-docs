# 議事メモ要約

## 手順

1. 議事メモ全文を読む
2. 次の 3 区分に整理する
   - **決定事項**
   - **担当者**（誰が何を担当するか）
   - **次のアクション**（期限があれば記載）
3. 元メモにない推測は書かない
4. 保存を依頼されたら `meeting-summary.md` としてプロジェクトルートに書き出す

## 出力フォーマット（Markdown）

```markdown
# 議事メモ要約

## 決定事項
- （箇条書き）

## 担当者
- （担当と内容）

## 次のアクション
- （期限付きであれば記載）
```
