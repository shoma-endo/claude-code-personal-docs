# 連絡メール下書き

## 手順

1. 元になる議事メモ（`議事メモ.txt`）または議事録サマリー（`議事録サマリー.md`）を読む
2. 下の **テンプレート（型）** に沿って本文を組み立てる
   - 並び順は **お礼 → 決まったことの要点（2〜3点）→ 次回のご案内**
3. 文体は **ですます調**。宛名・差出人・署名は `［　］` のプレースホルダにする
4. 元メモにない日程・数値・約束は **書かない**（推測しない）
5. 保存を依頼されたら `連絡メール下書き.md` としてプロジェクトルートに書き出す

## テンプレート（型）

```text
件名: ［会議名］のお礼と次回のご案内

［宛名］ 様

お世話になっております。［差出人名］です。
先日は［会議名］にご参加いただき、ありがとうございました。

決まったことの要点は以下のとおりです。

- ［要点1］
- ［要点2］
- ［要点3］

つきましては、次回のミーティングをご案内いたします。

- 日時: ［日時］
- 議題: ［議題］

ご不明点がありましたら、お気軽にご連絡ください。
引き続きどうぞよろしくお願いいたします。

［署名］
```
