## 読み方ガイド

講義中に口に出す機会が多く、スペルから読み方が分かりにくい用語の一覧です。スペル通りにアルファベットを読むだけのもの（CLI、API、MCP など）は省いています。

| 表記 | 読み方 | 補足 |
|---|---|---|
| `cc-secretary` | シーシー・セクレタリー | cc = Claude Code の略 |
| `npm` | エヌ・ピー・エム | 「ナム」とは読まない。Node Package Manager の略 |
| `Node.js` | ノード・ジェイエス | .js = JavaScript の略称 |
| `JSON`（settings.json など） | ジェイソン | JavaScript Object Notation の略 |
| `bash` | バッシュ | Bourne Again Shell。ターミナルのシェル名 |
| `zsh` | ズィーシェル | Z Shell。macOS のデフォルトシェル |
