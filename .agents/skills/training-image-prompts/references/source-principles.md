Use the following interpretation when designing training images.

## One Visual, One Message

Each image should carry one clear learning point. If a section needs multiple points, create multiple small visuals or a numbered process diagram instead of one crowded image.

## Reduce and Group Information

Remove details that do not help the learner complete the next action. Group related items closely, separate unrelated groups with whitespace, and avoid dense paragraphs inside the image.

## Guide Eye Flow

Arrange elements in the order learners should understand them. For Japanese business training pages and slides, default to top-left to bottom-right or left-to-right process flow.

## Use Readable Typography and Restrained Color

Use short labels, large text, high contrast, and a limited color system. Assign roles to colors instead of decorating with many colors:

- neutral: background and inactive structure
- primary accent: important action or current focus
- success accent: completed or recommended state
- warning accent: mistake, risk, or caution

## Keep Consistency Across Materials

When generating multiple images for the same training, repeat the same visual language: aspect ratio, line weight, color roles, label style, arrow style, and spacing density.

## Practical Translation for ChatGPT Image 2

Be explicit about:

- destination medium
- learning objective
- composition
- exact text
- color roles
- constraints

Avoid relying only on style words such as "beautiful", "cool", or "modern"; they do not specify the instructional job the image must do.
